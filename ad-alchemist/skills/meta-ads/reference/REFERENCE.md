## Meta Ads Ecosystem Overview

### Platform Hierarchy

```
Business Manager
└── Ad Account
    └── Campaigns (Objective level)
        └── Ad Sets (Targeting, budget, schedule)
            └── Ads (Creative, copy, CTA)
```

### Key Products

- **Ads Manager**: Primary interface for campaign management
- **Business Suite**: Unified inbox, scheduling, insights
- **Commerce Manager**: Product catalogs, shops
- **Events Manager**: Pixel, Conversions API, event tracking

### Advantage+ Suite (2026)

Meta's AI-powered campaign products:
- **Advantage+ Shopping Campaigns**: Automated e-commerce campaigns
- **Advantage+ App Campaigns**: Automated app install campaigns
- **Advantage+ Placements**: AI-optimized placement delivery
- **Advantage+ Audience**: Expanded targeting beyond defined audiences
- **Advantage+ Creative**: Automated creative enhancements

## 2026 Meta Ads Landscape

### Algorithm: Andromeda

Meta's current ad delivery system emphasizes:
1. **Creative signals** as primary targeting input
2. **Broad audiences** perform better than narrow targeting
3. **Conversion optimization** requires 50 events/week minimum
4. **First-party data** increasingly valuable post-iOS 14.5

### Key Trends

1. **Simplification wins**: Fewer campaigns with more budget each
2. **AI-powered everything**: Advantage+ products dominate
3. **Video-first**: Reels inventory growing fastest
4. **Privacy-centric**: Conversions API, first-party data crucial
5. **Creative diversity**: Multiple variations beat single "hero" creative

### Platform Distribution

| Placement | % of Inventory | Best For |
|-----------|---------------|----------|
| Facebook Feed | ~30% | All objectives |
| Instagram Feed | ~20% | Visual products, lifestyle |
| Reels | ~25% | Awareness, engagement |
| Stories | ~15% | Urgency, promotions |
| Other | ~10% | Retargeting, incremental reach |
